@startuml
title Оформление заказа еды

actor Клиент
actor Ресторан
actor Курьер

participant "Мобильное приложение" as App
participant "Система заказов" as System
participant "Платежная система" as Pay
database "База данных" as DB

Клиент -> App: Выбрать блюда
Клиент -> App: Оформить заказ

App -> System: Передать данные заказа

System -> Ресторан: Проверить заказ
Ресторан --> System: Заказ подтвержден

System -> Pay: Выполнить оплату
Pay --> System: Оплата выполнена

System -> DB: Сохранить заказ
DB --> System: Заказ сохранен

System -> Ресторан: Передать заказ на приготовление
Ресторан --> System: Заказ принят

Ресторан -> System: Заказ готов

System -> Курьер: Назначить доставку
Курьер --> System: Доставка принята

System --> App: Заказ оформлен
App --> Клиент: Показать номер заказа
@enduml
